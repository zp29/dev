# gulp webpack环境搭建

> 对各种前端需求搭建的环境,如果你用的上,可以考虑:star2:,以下的环境列表

### es6环境
  - webpack
  - babel
  - gulp
  - browser-sync
```
npm i
gulp --watch
```

```
npm i 
gulp
```

```
npm i 
gulp
```
